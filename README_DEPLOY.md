# 🚀 Быстрое развертывание CRM Hospital

## Минимальная инструкция для развертывания на чистом Linux сервере

### 📋 Что вам нужно:
- Linux сервер со статическим IP
- Root доступ
- 5-10 минут времени

---

## ⚡ Быстрый старт (3 команды)

```bash
# 1. Установите Docker
curl -fsSL https://get.docker.com | sudo sh

# 2. Загрузите проект на сервер (через git или scp)
git clone https://github.com/weae-g/CRM-military-hospital.git
cd CRM-military-hospital

# 3. Запустите!
chmod +x deploy.sh && ./deploy.sh start
```

**Готово!** Ваш сайт доступен по адресу: `http://ваш-ip-адрес`

---

## 🔧 Настройка перед запуском (опционально)

Если нужно изменить настройки:

```bash
# Отредактируйте конфигурацию
nano .env.production

# Важно: измените пароль базы данных!
POSTGRES_PASSWORD=ваш_безопасный_пароль
```

---

## 📝 Основные команды

```bash
./deploy.sh start    # Запустить
./deploy.sh stop     # Остановить
./deploy.sh restart  # Перезапустить
./deploy.sh logs     # Посмотреть логи
./deploy.sh status   # Статус сервисов
./deploy.sh backup   # Бэкап базы данных
```

---

## 🌐 Доступ к приложению

После запуска:
- **Сайт**: `http://your-server-ip`
- **API документация**: `http://your-server-ip/docs`

---

## 🔒 Безопасность (ВАЖНО!)

1. Измените пароль БД в файле `.env`
2. Настройте firewall:
   ```bash
   sudo ufw allow 80/tcp
   sudo ufw enable
   ```
