Инструкции

Для управления документами используйте форму выше для добавления новых записей. Каждый документ должен содержать отделение, имя, должность, звание и количество мест. Для редактирования существующего документа заполните поля новыми значениями и нажмите "Редактировать". Чтобы удалить документ, нажмите "Удалить" рядом с соответствующей записью.
